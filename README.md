# 我的 UE 游戏 Demo

这是一个基于虚幻引擎的多人在线游戏演示，展示了基本的联机对战功能。

## 如何运行多人游戏

游戏内置了控制台命令，你可以通过控制台快速启动或加入多人游戏。

> **打开控制台**：在游戏中按下 **`~`** 键（默认位于 Esc 键下方）即可打开控制台输入界面。

### 🟢 作为主机（创建房间）

如果你想作为主机创建游戏，等待其他玩家加入，请输入以下命令：
open FirstPersonMap_PvP?listen

### 🟢 作为客户端（加入房间）

如果你想作为客户端加入其他玩家，请输入以下命令：
open 127.0.0.1:7777
